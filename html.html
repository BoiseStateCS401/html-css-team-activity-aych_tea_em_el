Web Basics


Html

HTML functions as a kind of skeleton for the webpage. It contains the content and the structure and content of the page. HTML uses many opening and closing tags to put said content and information in the page. HTML calls these tags elements and as as barebones as it seems does a good job at containing the information.

Learn more at the w3schools introduction to HTML.

Or check out the Mozilla documentation on HTML.

CSS

CSS is the skin of the webpage. CSS handles the visuals and the overall aesthetic of a webpage.

Learn more at the w3schools introduction to CSS

Javascript

Javascript expands interactive functionality on the web, enabling animations and formatting for an intuitive experience.

Facts About the HTML

  • Which elements are block elements and which elements are inline? How do you know?
    • nav, div, p, headers, and lists default to block elements, while strong tags, line breaks, and links default to inline
  • What is the default margin of the body element?
    • 8px
  • What are the default font-size and font-weight of the h1 element?
    • 28.8px, BOLD
  • How long does it take to load your page?
    • ~81ms
  • How can you view the response status of the requested resources (e.g. images in this case)?
    • You can view these statuses through a browser's built in inspection tools. The response statuses are typically found on a "networking" tab.
